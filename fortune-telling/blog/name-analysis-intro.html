姓名學認為，每個字的筆劃數有其五行屬性和吉凶含義。名字的天格、人格、地格、外格、總格（五格）組合，會影響人生各個面向。

五格怎麼算？

天格 = 姓的筆劃+1｜人格 = 姓的筆劃+名第一字筆劃｜地格 = 名字所有字筆劃之和+1（單名+1）｜外格 = 總筆劃-人格+1｜總格 = 姓名所有字筆劃之和

哪些筆劃數比較好？

取名不只看筆劃吉凶，更重要的是配合八字的用神五行。命中缺水的人，名字帶水字旁或三點水最好。硬要用火字旁反而加重失衡。

改名真的有用嗎？

有的人改名後運勢確實改善，但這可能跟心理暗示也有關。改名至少做到兩件事：(1) 配合八字用神 (2) 五格都在吉數上。

取名注意事項
